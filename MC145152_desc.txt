MC145152 — это КМОП-синтезатор частоты с фазовой автоподстройкой частоты (ФАПЧ) и параллельным программированием делителей, выпускаемый в 28-выводном корпусе (DIP-28, SOIC-28, PLCC). Ниже представлена подробная распиновка микросхемы. [1, 2, 3]
Назначение выводов (Pinout)
•	1. \(f_{in}\) — Вход частоты ГУН (VCO). Сюда подается сигнал, который нужно стабилизировать.
•	2. \(V_{SS}\) — Общий провод (Земля).
•	3. \(V_{DD}\) — Питание (+3...9 В).
•	4. \(PD_{out}\) — Выход фазового детектора.
•	5, 6, 7. RA0, RA1, RA2 — Входы выбора коэффициента деления опорной частоты (R-counter).
•	8, 9. \(\Phi _{V}\), \(\Phi _{R}\) — Выходы фазового детектора для формирования сигнала ошибки петли ФАПЧ.
•	10. \(f_{V}\) — Выход буферизованного N-счетчика.
•	11–20, 22–25. N0–N13 — Входы данных N-счетчика (коэффициент деления основной частоты).
•	21. T/R — Вход управления сдвигом частоты (Передача / Прием).
•	26, 27. \(OSC_{out}\), \(OSC_{in}\) — Вывод и вход для подключения опорного кварцевого резонатора.
•	28. LD (Lock Detector) — Выход индикатора захвата частоты (высокий уровень при синхронизации). [1, 2, 3, 4, 5, 6, 7]
Таблица входов адреса опорного делителя (R)
Коэффициент деления (R-counter) устанавливается логическими уровнями на выводах RA0, RA1, RA2 (Выв. 5, 6, 7)

RA2	RA1	RA0	Коэффициент деления R
L	L	L	8
L	L	H	64
L	H	L	128
L	H	H	256
H	L	L	512
H	L	H	1024
H	H	L	1160* (или 2410 в зависимости от ревизии)
H	H	H	2048

где (H = Логическая 1 / Питание, L = Логический 0 / Земля)
